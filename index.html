🔧 Troubleshooting 401 Error

  • Get a fresh access token (they expire hourly)
  • Ensure token has Mail.Read permission
  • Remove any spaces before/after the token
  • Use Microsoft Graph API token, not other types
  • Check token format: starts with "eyJ..."
